# INSTALACJA

**Skopiuj i wklej do terminala**

```
git clone https://github.com/Croosheck/table-with-breadcrumb-challenge
```

**Następnie:**

```
cd .\table-with-breadcrumb-challenge\
```

```
npm install
```

**Dla VS Code:**

```
code -r .
```

**Po udanej instalacji:**

```
npm run dev
```

**Otwórz:** [http://localhost:5173/](http://localhost:5173/)
